# NewTenangKITA v0.7.1 — UX Clean-up

## Purpose

Keep the core NewTenangKITA elements while reducing unnecessary visible text. Empathy should be felt through flow, wording and action design — not long explanations on every screen.

## Changes

- Shortened Home and Welcome content
- Replaced long product compass text with compact trust cues
- Moved governance and prototype boundary details to Amanah Data and Status Prototaip
- Converted several explanation-heavy sections into expandable details
- Made guest-mode action rows directly tappable
- Simplified access-support wording without removing OKU/caregiver support
- Kept GitHub-only, no-database, no-personal-data boundary

## Not changed

- No real MyDigital ID login
- No PADU integration
- No real SARA, STR or BUDI95 retrieval
- No PriceCatcher live adapter
- No database or backend
- No scoring, false eligibility or fabricated prices
